# Streamlit Authentication App

A simple authentication demo app built with Streamlit, featuring user registration, login, and password reset functionality.

## How to Run

1. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Run the Streamlit app:**
   ```bash
   streamlit run app.py
   ```

3. **Access the app:**
   - The app will automatically open in your default web browser
   - If not, navigate to `http://localhost:8501`

## Features

- User registration with email validation
- Secure login with password hashing
- Password reset functionality with token-based system
- SQLite database for user storage

## Note

This is a demo application and should NOT be used in production environments.
